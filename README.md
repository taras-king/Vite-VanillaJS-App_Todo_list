# Todo List App

Простий та зручний менеджер завдань, побудований на Vite + Vanilla JavaScript.

## Стек технологій

- **Vite** — збірник проєкту
- **JavaScript** — логіка застосунку
- **HTML5 / CSS3** — розмітка та стилі

## ⚙Встановлення та запуск

1. Клонуй репозиторій:
```bash
   git clone https://github.com/taras-king/Vite-VanillaJS-App_Todo_list.git
```

2. Перейди у папку:
```bash
   cd Vite-VanillaJS-App_Todo_list
```

3. Встанови залежності:
```bash
   npm install
```

4. Запусти проєкт:
```bash
   npm run dev
```

## Функціонал

- Додавання нових завдань
- Позначення завдань як виконаних
- Видалення завдань

This App is deployed to gh-pages (https://bernhard759.github.io/Vite-VanillaJS-App_Todo_list/).
